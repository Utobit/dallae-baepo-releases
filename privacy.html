개인정보처리방침

최종 업데이트: 2026-04-23

Dallae PC Agent(이하 "본 서비스")는 사용자의 개인정보를 수집하지 않는 것을 기본 원칙으로 합니다.

1. 수집하는 개인정보

수집 항목: 없음. 본 서비스는 이름, 이메일, 위치, 기기 식별자 등 어떤 개인정보도 서버에 수집하지 않습니다.

2. 로컬 저장 데이터

아래 데이터는 사용자의 기기(PC) 내에만 저장되며, 외부로 전송되지 않습니다:

  • 설정값 (취침 모드, URL, 바로가기 이름 등)
  • 수면 기록 (취침/기상 시각)
  • 페어링 UUID (기기 내 한정, 변경 불가)

3. AI 챗봇 (고객지원)

웹사이트 또는 앱 내 AI 챗봇 사용 시:

  • 대화 내용은 개발자에게 직접 전달될 수 있습니다. (챗봇 시작 시 고지)
  • 대화 내용은 제3자 서비스에 판매·공유하지 않습니다.
  • AI 응답은 Anthropic Claude API를 경유합니다. (Anthropic 개인정보처리방침)

4. 자동 업데이트

업데이트 확인 시 GitHub Releases 서버에 HTTP 요청이 전송됩니다. 이 요청에는 현재 버전 번호만 포함되며 개인정보는 포함되지 않습니다.

5. 제3자 제공

수집된 개인정보가 없으므로 제3자에 제공할 정보가 없습니다.

6. 광고

웹사이트에 제3자 광고가 표시될 수 있으며, 해당 광고 제공자의 개인정보처리방침을 별도로 따릅니다.

7. 문의

개인정보 관련 문의는 웹사이트 내 AI 챗봇 또는 GitHub Issues를 통해 접수해 주세요.
