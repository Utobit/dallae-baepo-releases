설치 방법

1
PC 설치

PC 다운로드 버튼 클릭 → ZIP 압축 해제 → 폴더 안 DallaeBaepo.exe 실행.

2
Android APK 설치

APK 파일을 폰으로 전송 → "알 수 없는 앱 허용" → 설치.

3
모바일 연동

모바일 앱 설정 탭 → PC IP 주소 입력 (PC 앱 화면에서 확인).
